Linux学习记录（二）实用指令：系统和文件目录类

一、指定运行级别

七个运行级别

0. 关机
1. 单用户
2. 多用户无网络
3. 多用户有网络
4. 保留
5. 图形界面
6. 重启

运行级别配置文件

/etc/inittab CentOS 7中已废弃，使用target管理

命令（CentOS 6）

init[012356]

命令（CentOS 7）

systemctl [command] [unit.target]

command:

1. get-default :取得当前的target
2. set-default :设置指定的target为默认的运行级别
3. isolate :切换到指定的运行级别
4. unit.target :运行级别

运行级别：

0.shutdown.target
1.emergency.target
2.rescure.target
3.multi-user.target
4.无
5.graphical.target
6.无

二、帮助指令

man基本语法

man [命令/配置文件]

示例

man ls

helo基本语法

help [命令]

示例

help cd

百度/谷歌

三、文件目录类

pwd指令

显示当前目录的绝对路径
pwd

ls指令

ls [选项] [目录/文件]
常用选项
-a -l

CD指令

跳转目录
cd 参数
常用参数
~ - .. /
绝对路径和相对路径
绝对路径：从根目录/开始定位 /home/lwq
相对路径：从当前工作目录开始定位 ../lwq

mkdir指令

创建目录
mkdir [选项] 要创建的目录
示例：mkdir /home/dog
常用选项
-p创建多级目录
示例：mkdir -p /home/dog/cat/niuzi

rmdir指令

rmdir [选项] 要删除的空目录
示例：rmdir /home/dog
如果删除非空目录则需要使用 rm -rf 要删除的目录

touch指令

创建空文件
touch 文件名
示例：touch niuzi.c touch file{1,2,3}

copy指令

拷贝指定文件到指定目录
cp [选项] source dest 示例：cp file1 ~/learn/test常用选项：-r递归复制整个文件夹（包括文件） 示例：cp -r learn ~/learn/test其中learn为文件夹 忽略覆盖提示：\cp……`

rm指令

移除文件或目录
rm [选项] 要删除的文件或目录
示例：rm /home/lwq/learn/file1 rm /home/lwq/learn/*
常用选项：
-r：递归删除整个文件夹
-f：强制删除，不提示
示例：rm -rf /

mv指令

移动或重命名文件与目录
mv oldNameFile newNameFile重命名
mv /temp/moveFile /targetFolder移动文件

cat指令

查看文件内容
cat [选项] 要查看的文件
常用选项：
-n：显示行号
示例：cat -n file2
一般会带上管道命令 |more

more指令

基于VI编辑器的文本过滤器，它以全屏幕的方式俺页显示文本文件的内容
more 要查看的文件
快捷键：

空格：向下一页
Enter：像下一行
q：离开
ctrl+f：向下一屏
ctrl+b：返回上一屏
=：输出当前行号
f：输出文件名当前行号

less指令

分屏查看文件内容，并不是一次将整个文档加载后才显示，而是根据现实需要按需加载，常用于大文件
less 要查看的文件

>指令和>>指令

>输出重定向和>>追加
ls -l >文件列表内容写入文件中（覆盖）
ls -l >>文件列表内容追加到文件末尾
cat 文件1 > 文件2将文件1的内容覆盖到文件2
echo "内容" > 文件将”内容”写入到文件中

echo指令

输出内容到控制台
echo [选项] [输出内容]
示例：
echo $PATH输出当前环境变量路径
echo "hello"输出文本：hello

head指令

显示文件开头部分内容，默认前10行
head 文件
head -n 5 文件

tail指令

输出文件中尾部内容，默认输出后10行
tail 文件
tail -n 5 文件
tail -f 文件实时追踪该文档的所有更新

ln指令

软链接/符号链接，类似win的快捷方式，主要存放了链接其他文件的路径
ln -s [原文件/目录] [软链接名]给源文件创建一个软链接
删除软链接：rm -rf 软链接末尾不要带/

history指令

查看已经执行过的历史命令，也可以执行历史命令
history所有
history 10后10条
! 111执行编号为111的历史指令
